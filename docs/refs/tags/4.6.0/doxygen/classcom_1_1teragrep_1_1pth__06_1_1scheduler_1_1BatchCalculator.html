Detailed Description

Batch

Contains the necessary operations to form a Spark batch. It consists of Archive and/or Kafka data. Each batch is constructed from a RangeProcessor, which in turn consists of multiple BatchUnits. Each of the slices contain the actual data.

Author
Eemeli Hukka, Mikko Kortelainen
